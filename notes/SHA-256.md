##  What is SHA-256 Hash?

SHA-256 is used to convert data into a **unique set of characters (a hash)**.

###  How it works (simple idea)

- Takes input (text, image, file, etc.)
- Breaks the input into smaller parts
- Mixes the data again and again using multiple rounds
- Produces a **unique fixed-length string**

###  Key Idea
> SHA-256 takes data, breaks it, and repeatedly mixes it to produce a unique string.

###  Important Points

- Same input → Same hash  
- Different input → Completely different hash  
- Output is always the same length (64 characters)  
- It is **one-way** (cannot get original data back)

### 🔄 Flow

> Input → Break → Mix (many times) → Unique Hash



### 🚀 Use Cases

- Security
- Password storage
- File verification
- Cybersecurity systems
